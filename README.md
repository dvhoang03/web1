## 1. Yêu cầu hệ thống
- Node.js 
- MongoDB
## 2. Cài đặt
### Backend
Clone dự án về máy:
 

git clone https://github.com/dvhoang03/web1

 
cd backend
npm install
Khởi động server:
 
npm start
### Frontend
Điều hướng tới thư mục frontend và cài đặt:

cd frontend
npm install
Chạy frontend:

npm start
###  Sử dụng
Mở trình duyệt và truy cập http://localhost:3000 để bắt đầu trải nghiệm website bán hàng.
